## üö® **Troubleshooting**

### **Common Issues**

#### **Dashboard Not Loading**
- Check admin authentication
- Verify server is running
- Clear browser cache
- Check browser console for errors

#### **AI Chat Not Responding**
- Verify AI services are running
- Check `/api/ai/free-status` endpoint
- Review AI configuration in `.env`
- Restart AI services if needed

#### **Metrics Not Updating**
- Check real-time updates are enabled
- Verify API endpoints are accessible
- Review browser network tab for errors
- Refresh dashboard page

#### **Health Check Failing**
- Verify all AI systems are running
- Check system resources (CPU/memory)
- Review error logs
- Restart services if necessary
