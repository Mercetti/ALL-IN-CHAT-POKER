ğŸ”§ Troubleshooting Common Issues

Solutions to frequently encountered problems

ğŸ¯

Overlay Issues

Overlay appears empty

Check browser console for socket connection errors. Ensure backend URL is reachable and channel is set correctly.

OBS lag or performance issues

Try lowering browser source FPS to 30 or disabling hardware acceleration. Check your system resources.

ğŸ¤–

Bot Problems

Bot not joining channel

Verify BOT_JOIN_SECRET matches server secret. Check if bot account follows your channel. Ensure betting is open.

Commands not working

Check bot console for errors. Verify OAuth token is valid and not expired. Ensure target channels are configured.

ğŸ”

Authentication Issues

OAuth redirect mismatch

Update Twitch app redirect URL to exactly match your deployed login URL, including protocol.

Login not working

Clear browser cache and cookies. Check if JWT token is stored properly in localStorage.
