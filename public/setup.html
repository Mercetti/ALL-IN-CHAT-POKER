Setup Guide

Overlay + Bot Quickstart

Drop the overlay into OBS, point it at the live backend, then let chat join with !join or bet with !bet <amount>. Use the bot to mirror chat into the game without leaving Twitch.

Open OBS Overlay Admin Dashboard

1) OBS Browser Source

Add a browser source and point it at the overlay URL.

  1. URL: https://all-in-chat-poker.fly.dev/obs-overlay.html (or your domain)
  2. If you're not logged in on that machine, append ?channel=<your_twitch_login>
  3. Width: 1920 · Height: 1080 (or your canvas); FPS: 60; enable hardware accel.
  4. Place above your background, below webcam.

If you’re logged in as the streamer, the overlay will auto-bind to your channel.

2) Bot Environment

Run the Twitch bot so chat can sit with !join / !bet.

Required env
  • BOT_USERNAME – Twitch bot username
  • BOT_OAUTH_TOKEN – Twitch IRC token (oauth:...)
  • TARGET_CHANNELS – comma list (no #), or leave empty and use backend-driven channels
  • ADMIN_TOKEN – backend admin bearer
  • BOT_JOIN_SECRET – must match server secret
  • BACKEND_URL – e.g. https://all-in-chat-poker.fly.dev
Start locally
node bot/bot.js

3) Bot Commands (chat)

  • !join – seats user with min bet if betting is open
  • !bet <amount> – places a bet for the user
  • !start / !startnow – admin-only round control
  • !leaderboard, !rules, !status – utility

Ensure BOT_JOIN_SECRET matches the server secret so chat bets are accepted.

4) Twitch OAuth Redirect

Configured to https://all-in-chat-poker.fly.dev/login.html. Match this in the Twitch app settings (add your custom domain too). Overlay and admin require a valid user token.

5) Admin Tips

  • Use the Admin dashboard to start rounds or reset the game.
  • Overlay buttons on the admin header open both overlay pages.
  • OBS overlay mirrors player state; use !join to populate quickly.

6) Troubleshooting

Overlay empty?
Check socket connection in the browser console; ensure the backend URL is reachable and the channel is set (query param or logged-in streamer).

Bot not joining?
Verify BOT_JOIN_SECRET matches server and betting is open.

OAuth redirect mismatch?
Update Twitch app redirect to exactly match the deployed login URL.

OBS lag?
Lower browser source FPS or disable hardware accel toggle.
